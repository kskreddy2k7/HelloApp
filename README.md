# 👋 Hello App (Java)

A basic Java application that demonstrates the **fundamentals of Java programming** by printing a simple message.
This project is part of my initial learning in Java and helps build a strong foundation.

---

## 📌 Project Overview

The **Hello App** is a beginner-level program designed to understand:

* Java syntax
* Program structure
* Compilation and execution process

---

## 🚀 Features

* 🔹 Simple and beginner-friendly
* 🔹 Demonstrates basic Java program structure
* 🔹 Easy to compile and run

---

## 🧠 Concepts Used

* ✔ Java Basics
* ✔ Main Method

## ▶️ How to Run

1. Clone the repository:

```id="aof4uv"
git clone https://github.com/kskreddy2k7/HelloApp.git
```

2. Navigate to the project:

```id="mrm0yk"
cd HelloApp
```

3. Compile the program:

```id="hhq8rd"
javac Hello.java
```

4. Run the program:

```id="oohg18"
java Hello
```
